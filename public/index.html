Help & Support

How to Host a Session

  1. Click "Host Session" on the main screen
  2. Enter a unique Room ID or generate one randomly
  3. Choose your preferred quality and frame rate settings
  4. Click "Start Sharing Session"
  5. Share the Room ID with viewers you want to connect

How to Join a Session

  1. Click "Join Session" on the main screen
  2. Enter the Room ID provided by the host
  3. Click "Connect to Session"
  4. Wait for the screen share to load
  5. Click "Enable Control" to start remote control

Keyboard Shortcuts

F11 - Toggle Fullscreen
Ctrl+Q - Disconnect
Space - Toggle Control
Esc - Exit Fullscreen

Troubleshooting

Screen sharing not working: Make sure the host has proper screen recording permissions enabled.
Remote control not responding: Ensure the host has accessibility permissions enabled for mouse/keyboard control.
Poor connection quality: Try reducing the quality and frame rate settings.
Cannot connect to room: Verify the Room ID is correct and the host session is active.
